# Task 5 - Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis (EDA) on a sales dataset to identify patterns, trends, relationships, and business insights using Python.

## Dataset
Sales Data Sample (Cleaned Dataset)

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

## Analysis Performed

### Data Exploration
- Dataset overview
- Data types inspection
- Statistical summary
- Missing value analysis

### Visualizations
- Sales Distribution Histogram
- Sales Boxplot
- Revenue by Product Line
- Top Countries by Revenue
- Correlation Heatmap
- Pairplot
- Quantity Ordered vs Sales Scatter Plot

## Key Findings
- USA generated the highest revenue among all countries.
- Classic Cars was the highest-performing product line.
- Several high-value sales outliers were identified.
- Quantity Ordered showed a positive relationship with Sales.
- Correlation analysis helped identify relationships among numerical variables.
- Visual analysis revealed important sales patterns and trends.
